# Talisman Studio – Privacy Policy & Terms of Service

This directory contains the static site files for the Privacy Policy & Terms of Service of Talisman Studio.

## Deployment

1. Ensure you have Netlify CLI installed: `npm install -g netlify-cli`  
2. From project root run: `netlify deploy` or `netlify deploy --prod`  
3. Ensure `netlify.toml` is configured (see config file).  
4. The policy page is available at `/privacy` (redirects to `privacy-policy.html`).

## Updating the Policy

- Update the “Last updated” date at the top of `privacy-policy.html` when making changes.  
- Review your data collection, usage, and rights sections periodically.  
- Re-deploy to make changes live.
